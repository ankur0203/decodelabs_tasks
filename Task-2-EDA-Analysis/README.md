 Task 2 - Exploratory Data Analysis (EDA)

## Dataset
- Dataset: E-Commerce Sales Data
- Total Orders: 1,200
- Period: Jan 2023 – Jun 2025
- Variables: 14
- Tool Used: Microsoft Excel

## Key Metrics
- Total Revenue: ₹1,264,762
- Average Order Value: ₹1,054
- Missing Values: 309 (CouponCode column)

## Key Observations
- **Revenue Trend:** Highest in Jun 2024 (₹68,069), 
  Lowest in Apr 2023 (₹27,752)
- **Product Performance:** Printer leads in orders (181), 
  Laptop has highest avg order value (₹1,111)
- **Order Status Alert:** 49.7% orders Cancelled or 
  Returned — major business concern
- **Outliers:** 8 high-value outliers found (₹3,334–₹3,456)
- **Referral Source:** Instagram top channel (259 orders)
- **Payment Method:** Online payment most popular (21.5%)
